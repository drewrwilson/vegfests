vegfests
========

Starting to map vegetarian festivals. Open [where.geojson](https://github.com/drewrwilson/vegfests/blob/master/where.geojson) to view it.

## format

{
  "type": "Feature",
  "properties":
  {
    "name" : "",
    "2014 date" : "",
    "2015 date" : "",
    "city" : "",
    "location" : "",
    "website" : ""
  },
  "geometry":
  {
    "type": "Point",
    "coordinates": [ -x , y ]
  }
},
